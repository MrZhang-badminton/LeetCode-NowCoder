# LeetCode and NowCoder

### 项目描述

+ 本项目主要包含一些常见的基础、高级数据结构
+ 同时也包含了LeetCode和NowCoder上的算法

### 项目模块

+ advanced
    + 一些高级数据结构

+ basic
    + 基础数据结构与算法
        + 线性结构
        + 字符串
        + 树
        + 图
        + 排序 +还有一些常见的算法

+ oj
    + lc
        + LeetCode中的算法
    + nc
        + NowCoder中的算法

+ utils
    + Node链表节点、TNode树节点
    + 工具类
